# dynatrace_environment_ip_allowlist (Resource)

-> **Dynatrace SaaS only**

-> To utilize this resource, please define the environment variables `DT_CLIENT_ID`, `DT_CLIENT_SECRET`, `DT_ACCOUNT_ID` with an OAuth client including the following permissions: **View environments** (`account-env-read`) and **Manage environment settings** (`account-env-write`).

## Dynatrace Documentation

- PUT IP allowlist - https://docs.dynatrace.com/docs/dynatrace-api/account-management-api/environment-management-api/put-ip-allowlist
- GET IP allowlist - https://docs.dynatrace.com/docs/dynatrace-api/account-management-api/environment-management-api/get-ip-allowlist

## Resource Example Usage

```terraform
variable "ENVIRONMENT_ID" {
  description = "The environment ID."
  type        = string
}

resource "dynatrace_environment_ip_allowlist" "test" {
  environment_id         = var.ENVIRONMENT_ID
  enabled                = true
  allow_webhook_override = false

  allowlist {
    name     = "office"
    ip_range = "10.0.0.0/8"
  }
}
```
